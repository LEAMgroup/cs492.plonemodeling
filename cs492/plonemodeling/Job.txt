Job Tests
-----------------

This is a DocTest file, using the Python doctest format.


Create a Job object and put it in a folder

    Create the object::

    >>> from zope.component import createObject
    >>> container = createObject('cs492.plonemodeling.job')

    Verify it's type::

    >>> container.portal_type
    'cs492.plonemodeling.job'

    Give it a title::

    >>> container.setTitle('Test Title')
    >>> container.Title()
    'Test Title'

    Put it in our base folder::

    >>> from plone.dexterity.utils import addContentToContainer
    >>> cobj = addContentToContainer(folder, container)

    >>> cobj
    <Job at /plone/Members/test_user_1_/test-title>

    >>> folder['test-title'].Title()
    'Test Title'
